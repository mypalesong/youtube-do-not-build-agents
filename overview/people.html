주요 인물 ​

이 발표에 등장하거나 언급되는 인물들을 정리한다.

발표자 ​

Barry Zhang -- Anthropic 엔지니어 ​

Anthropic에서 Agent Skills 프로젝트의 핵심 구축자 중 한 명이다. 발표에서 그는 "세무 코드 처리라면 Mahesh보다 Barry가 낫다"는 농담의 주인공이자, 전문성이 범용 지능보다 실무에서 강하다는 논지를 상징하는 인물로 등장한다.

Zhang은 AI Engineer 컨퍼런스를 비롯한 공개 행사에서 에이전트 설계와 Claude Code의 엔지니어링 의사결정을 자주 공유해왔다. 실제로 Claude Code, Claude Agent SDK, 그리고 Skills 시스템의 설계 논의에 깊숙이 참여한 것으로 알려져 있다.

Mahesh Murag -- Anthropic 엔지니어 ​

Anthropic의 엔지니어로, Skills 생태계와 내부 시스템 구축에 관여했다. 발표에서는 Zhang이 농담 삼아 "300 IQ 수학 천재"라고 소개하는 역할을 맡으며, 순수 지능만으로는 실무를 해결하지 못한다는 비유의 반대편에 선다.

Murag은 Model Context Protocol(MCP) 관련 커뮤니티에서도 활동해왔으며, MCP와 Skills의 역할 분담(연결 vs 전문성)을 명확히 설명하는 장면은 발표의 기술적 하이라이트 중 하나다.

발표에서 언급된 조직 및 프로젝트 ​

Anthropic ​

Claude 모델 시리즈와 Claude Code를 만든 AI 안전 연구 기업. 이 발표의 맥락에서는 2025년 10월 Agent Skills를 공식 출시하고, 5주 만에 수천 개의 스킬이 생성되는 생태계를 형성한 주체다.

Claude Code ​

Anthropic의 첫 번째 코딩 에이전트이자, 발표자들이 **"코드는 디지털 세계의 범용 인터페이스"**라는 확신을 얻은 실험대. 발표는 Claude Code를 만들면서 관찰한 경험이 Skills의 설계로 이어졌음을 반복적으로 강조한다.

Cadence ​

생명과학/바이오인포매틱스 영역의 스킬을 구축한 파트너사. EHR 데이터 분석과 과학 연구를 위한 기초 스킬의 사례로 언급된다.

Browserbase (Stagehand) ​

웹 탐색/자동화 분야의 서드파티 스킬 제공자. Browserbase가 만든 Stagehand 스킬은 에이전트가 브라우저를 다루는 작업을 표준화된 방식으로 수행하게 해준다.

Notion ​

워크스페이스 이해와 연구를 위한 Notion 스킬을 출시한 파트너. 기업 내부 문서 검색, 작성, 분석 워크플로를 스킬로 패키지화한 사례다.

Model Context Protocol (MCP) ​

Anthropic이 개방형 표준으로 공개한 프로토콜. 발표에서 MCP = 외부 세계와의 연결, Skills = 전문성의 담지체라는 구도로 설명된다. 둘은 대체 관계가 아니라 서로를 보완한다.

Fortune 100 기업들 (익명) ​

발표에서는 이름을 특정하지 않으나, 대형 기업의 개발자 생산성 팀이 사내 코드 스타일과 빌드/배포 관행을 스킬로 인코딩하고 수천 명의 엔지니어에게 제공하는 사례가 언급된다. 또한 법무, 재무, 회계 부서의 비개발자 직군이 자신의 절차적 지식을 스킬로 만들고 있다는 관찰도 함께 제시된다.
